# 🚀 Minha Primeira Experiência com .NET CLI

Nesta missão, deixei de ser apenas um usuário de pastas e me tornei um desenvolvedor que utiliza o terminal para criar e executar aplicações.

## 🛠️ Comandos de Construção Utilizados

* `dotnet new console`: utilizado para criar a estrutura inicial de um projeto em C#.
* `dotnet build`: responsável por compilar o código, transformando em um executável.
* `dotnet run`: utilizado para executar o programa diretamente pelo terminal.

## 📦 Estrutura Gerada

Arquivos principais criados automaticamente pelo .NET:

1. `Program.cs`: arquivo onde está o código principal da aplicação.
2. `SistemaExpert.csproj`: arquivo de configuração do projeto.
